Bienvenido a mi Página Web

Esta es una página de ejemplo que muestra la estructura básica utilizando unicamente HTML.
Aunque en el desarrollo de la pagina hablemos de las 3 herramientas mas utilizadas para crear paginas web, no se debe ignorar ni dejar de tener en cuenta el hecho de que no se ha implementado el uso de CSS ni JS.

HTML: La estructura

¿Qué es?

HTML son las siglas de Markup Language (Lenguaje de Marcado de Hipertexto). Es un lenguaje de marcado que se utiliza para crear páginas web. A través de etiquetas y elementos, define la estructura y el contenido de una página, como encabezados, párrafos, imágenes, vídeos, enlaces y formularios. Los navegadores web interpretan este código y lo presentan al usuario de forma visual y organizada.

Es importante porque:

  1. Define la estructura de una página web: Es el esqueleto que da forma al contenido, organizándolo en secciones, títulos, párrafos, listas, etc.
  2. Permite insertar contenido multimedia: Imágenes, vídeos, audios y otros elementos se integran a la página mediante HTML.
  3. Crea enlaces entre páginas: Los hipervínculos (links) permiten navegar por la web y conectar diferentes páginas.
  4. Es compatible con todos los navegadores: Al ser un estándar web, funciona en cualquier navegador y dispositivo, asegurando una visualización consistente.
  5. Facilita la accesibilidad web: El uso correcto de etiquetas HTML contribuye a que las páginas sean accesibles para personas con discapacidades.
  6. Sirve como base para otros lenguajes web: CSS y JavaScript se complementan con HTML para dar estilo y dinamismo a las páginas.

CSS: El estilo

¿Qué es?

CSS son las siglas de Cascading Style Sheets (Hojas de Estilo en Cascada). Es un lenguaje de marcado que se utiliza para dar estilo a las páginas web. Controla la apariencia visual de los elementos HTML, como colores, tipografías, tamaños, márgenes, alineaciones y mucho más.

Es importante porque:

  1. Separa el contenido de la presentación: Permite modificar el diseño sin afectar la estructura de la página, facilitando el mantenimiento y la reutilización de código.
  2. Mejora la usabilidad y la experiencia de usuario: Un diseño atractivo y organizado hace que la navegación sea más intuitiva y placentera.
  3. Crea diseños consistentes: Se pueden aplicar estilos uniformes a todas las páginas de un sitio web, logrando una identidad visual coherente.
  4. Permite diseños responsivos: Las páginas se adaptan a diferentes tamaños de pantalla (computadoras, tablets, smartphones), optimizando la experiencia en cualquier dispositivo.
  5. Reduce el tiempo de carga: Al separar el estilo del contenido, las páginas web se cargan más rápido.
  6. Facilita la accesibilidad: Se pueden ajustar los estilos para personas con discapacidades visuales o auditivas.

JS: El dinamismo

¿Qué es?

JavaScript (JS) es un lenguaje de programación que se utiliza para agregar interactividad y dinamismo a las páginas web. Permite que las páginas respondan a las acciones del usuario, creando experiencias más atractivas y funcionales.

Es importante porque:

  1. Interactividad: Permite crear elementos que reaccionan a las acciones del usuario, como botones, menús, formularios, juegos y animaciones.
  2. Dinamismo: Actualiza el contenido de la página sin necesidad de recargarla, mejorando la fluidez y la experiencia del usuario.
  3. Efectos visuales: Crea animaciones, transiciones y otros efectos visuales que enriquecen el diseño y captan la atención del usuario.
  4. Aplicaciones web: Permite desarrollar aplicaciones web complejas, como juegos en línea, tiendas virtuales y herramientas interactivas.
  5. Integración con otros lenguajes: Se complementa con HTML y CSS para crear experiencias web completas y dinámicas.
  6. Soporte en todos los navegadores: Es compatible con los navegadores web modernos, asegurando un funcionamiento uniforme.
